more能做什么

more会显示一个屏的内容，在左下角还显示了文件的百分比,如果按空格键，文件的下一屏辉县市，如果按回车则显示下一行内容，如果输入’q’，结束现实。

more的基本用法

  • more filename
  • command | more
  • more < filename
